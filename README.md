### 👋 About Me

Hey, I'm Ivan. I am a software engineer based out of Red Deer, Alberta.

With over 8 years of programming experience, my focus spans across full-stack web development, backend architecture, and practical AI integration. I hold a Computer Programming Diploma from Red Deer Polytechnic and a Software Engineering Certificate from IT Step Academy.

I specialize in building robust enterprise solutions from the ground up. Whether I'm developing highly responsive web applications, designing scalable relational databases, or integrating local machine learning models into offline tools, I care about writing clean, efficient code that actually solves real-world problems. 

**What I spend my time on:**
* 💻 Developing dynamic web applications and end-to-end full-stack platforms.
* 🧠 Architecting scalable backend systems, APIs, and clean database schemas.
* 🤖 Engineering high-performance desktop applications and practical AI integrations.
